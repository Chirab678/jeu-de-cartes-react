# Jeu de Cartes React

Un projet React permettant d'échanger des cartes en les sélectionnant.

## Fonctionnalités
- Ajout de nouvelles cartes
- Sélection et échange de 2 cartes
- Design responsive

## Installation
```bash
git clone https://github.com/votre-utilisateur/jeu-de-cartes-react.git
cd my-react-app
npm install
npm run dev
